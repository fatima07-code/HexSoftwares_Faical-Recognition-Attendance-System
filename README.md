# 👤 Facial Recognition Employee Attendance System

A Python-based AI-powered attendance system that uses facial recognition technology (DeepFace + OpenCV) to automatically mark employee attendance without manual input or biometric devices.

---

## 🚀 Features

- 🔐 Employee Login System
- 👤 Employee Registration with Face Capture
- 📸 Automatic Face Recognition using DeepFace
- 🧠 Real-time webcam-based detection
- ✔ Automatic Attendance Marking
- 📊 Attendance stored in CSV file
- 🗂 SQLite Database for employee records
- 🖥 GUI-based interface using Tkinter
- 🚫 Prevents duplicate attendance in a single day
- 🎯 Popup notification on successful attendance

---

## 🛠️ Technologies Used

- Python 3.x
- OpenCV
- DeepFace
- Tkinter (GUI)
- SQLite3 (Database)
- Pandas
- NumPy
